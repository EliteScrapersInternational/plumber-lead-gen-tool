# 🪠 Plumber Lead Gen AI 

An automated lead discovery engine designed for Marketing Agencies and SEO Specialists.

## 🚀 What it does
This tool identifies plumbing businesses across multiple cities and analyzes their web presence to find "invisible" businesses that need professional help.

### Features:
- **Bulk City Scraping:** Input a list of cities and get results for all of them at once.
- **Opportunity Detection:** Flags businesses without websites or those relying solely on social media.
- **Export Ready:** Seamlessly download leads as Excel, CSV, or JSON.

## 🛠️ Tech Stack
- **Node.js**
- **Apify SDK**
- **Google Search Scraper Engine**
